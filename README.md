# Hi, I'm Mahmoud Abujadallah 👋

**PhD researcher in AI for Software Engineering @ École de technologie supérieure (ÉTS), Montréal.**

I study how AI coding agents behave in the wild — agentic pull requests, why their fixes get merged or rejected, and the trustworthiness of AI-generated code. I build the open datasets and tools that make that research possible.

🌐 **Portfolio & datasets → [mabujadallah.github.io](https://mabujadallah.github.io/)**

---

### 🔬 What I'm working on
- Mining millions of GitHub pull requests to measure how AI coding agents (Claude Code, Cursor, Copilot, Devin) actually perform.
- Understanding *why* agent-authored fixes get rejected — and what makes them mergeable.
- Open-source LLMs for evaluation tasks: automated essay scoring and Arabic NLP.
